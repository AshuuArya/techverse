Keyboard tips and tricks

Keyboard Shortcuts

  • Open the task manager directly: If you want to bypass the interrupt that happens when pressing Ctrl + Alt + Del and jump right to the task manager, follow these steps:
    • Press Ctrl + Shift + Esc.
  • Use Spotlight: To leverage macOS' Spotlight for efficient searching, follow these steps:
    • Press Cmd + Spacebar.
    • Type the first 2-3 letters of the app's name or enter your search query.
  • Interrupt all processes: To interrupt all processes on a PC or summon the Force Quit dialog box on macOS, follow these steps:
    • On PC: Press Ctrl + Alt + Delete.
    • On macOS: Press Cmd + Shift + Esc.
  • Cycle through open windows: To cycle through currently open windows quickly and efficiently, follow these steps:
    • On PC: Press Alt + Tab (Alt + Shift + Tab to cycle backwards).
    • On macOS: Press Cmd + Tab.
  • Launch programs with your own hotkeys: To set up custom hotkeys for launching programs in Windows, follow these steps:
    • Right-click the shortcut to any application.
    • Go to Properties and navigate to the Shortcut tab.
    • In the "shortcut key" field, type your preferred launch combo.
    • (Optional) Click on "advanced" options to set it to run as an Administrator.
  • Close the current program: To quickly close the current program running on your computer, follow these steps:
    • Press Alt + F4.
    • This shortcut saves you time by bypassing the need to manually click the "X" button.
    • Be cautious, as people may use Alt + F4 as a joke to trick you into closing what you're doing.
  • Minimize all windows: To minimize all open windows and quickly access your desktop, follow these steps:
    • Press Windows key + D.
    • This action minimizes all windows at once, saving time compared to minimizing each window individually.
    • While Windows + M offers similar functionality, there is no undoing, making Windows + D the preferred approach.
    • In macOS, leverage Mission Control for similar desktop management tasks.
  • You can use the menu key on a keyboard to right-click: Utilize this keyboard shortcut to quickly right-click on an item in Windows:
    • Press the "menu" key on your keyboard.
    • This key functions as a right-click, providing contextual options for the selected item.
    • Note that not all keyboards have this key.
  • Close the current window/tab: Save time by closing the current window or tab with this shortcut:
    • Press Ctrl + W.
    • This shortcut instantly closes the active window or tab, eliminating the need to manually click the "X" button.
    • Exercise caution when using this shortcut to avoid accidentally closing important windows or tabs.
  • Bring up the system information window: Quickly access the System Information panel with this keyboard shortcut:
    • Press Windows + Pause/Break.
    • This shortcut opens the System Information window without the need for manual navigation, saving time.
    • The Pause/Break key may have limited use otherwise, making this shortcut a convenient option.
  • Use your keyboard to launch programs on the Windows taskbar: Easily open programs pinned to the Windows taskbar using this shortcut:
    • Press the Windows key + the number corresponding to the position of the program on the taskbar.
    • For example, if Chrome is the second icon on your taskbar, pressing Windows key + 2 opens Chrome.
  • Use your keyboard to navigate system tray icons: Streamline navigation of system tray icons with this keyboard shortcut:
    • Press the Windows key + B.
    • This shortcut automatically selects the system tray area, allowing you to navigate using the arrow keys and open items with Enter.
  • Enable copy and paste in the Windows command prompt: Enhance functionality in the Windows command prompt with this setting adjustment:
    • Right-click the command prompt shortcut and select Properties.
    • In the Options tab, check the box next to "QuickEdit mode".
    • With QuickEdit mode enabled, you can select text with left-click, copy with Enter, and paste with right-click within the command prompt.
